== Installation

You don't need this source code unless you want to modify the gem. If
you just want to use the EasyPost Ruby bindings, you should run:

  gem install easypost

== Requirements

* Tested with Ruby 1.9.3. Other versions will likely work but have not been tested.
* typhoeus, json

== Documentation

Please see http://geteasypost.com/docs for the most up-to-date
documentation.
