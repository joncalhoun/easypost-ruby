== Installation

You don't need this source code unless you want to modify the gem. If
you just want to use the EasyPost Ruby bindings, you should run:

  gem install easypost

== Requirements

* Ruby 1.8.7 or above. (Other versions may work but are not supported)
* typhoeus, json
